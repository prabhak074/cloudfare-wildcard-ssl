Configuration

To use wildcard in cloudfare, follow these steps:

  1. First we must configure cloudfare as a dns.
  2. then we activate wildcard ssl.

Cloudfare DNS configuration:

To use CLoudfare DNS, follow these steps:

  1. First find the domain registrar for our case it is amazon route 53.
  2. then go to cloudfare dashboard and you will see add site.
  3. follow the add site documentaion from cloudfare https://developers.cloudflare.com/dns/zone-setups/full-setup/setup for detailed instructions on specific features.
For further references use below links:
 https://developers.cloudflare.com/fundamentals/get-started/setup/add-site/
 https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/domain-name-servers-glue-records.html#domain-name-servers-glue-records-adding-changing

Cloudfare wildcard configuration:

Our software allows for configuration settings to customize your experience. To configure the software, follow these steps:

  1. Create CloudFlare API credentials.
  2. Install Certbot and the CloudFlare DNS authenticator plugin.
  3. Run Certbot with the CloudFlare Authenticator.
  4. Automatic renewals.
